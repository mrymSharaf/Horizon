## About

**Horizon** is a travel-themed web application designed to inspire and guide users in exploring destinations around the world.It provides an engaging interface where users can browse travel ideas and view destination details.
This project was built to combine functionality with creativity offering a smooth user experience while showcasing modern web design practices.  
The motivation behind Horizon is to create an app that helps with travel discovery.

---

## Features  

- **User Authentication** ‚Äì secure sign-up, login, and profile management  
- **Image Uploads** ‚Äì add and preview travel images with **Cloudinary** integration  
- **Interactive Maps** ‚Äì explore locations dynamically using **Leaflet API**  
- **Animations** ‚Äì engaging travel-themed **Lottie animations** for a modern experience  
- **Dynamic Content** ‚Äì AJAX/jQuery-based updates without page reloads  
- **Follow System** - keep up with your friends updates
- **View Likes** - browse posts that a user has liked directly from their profile
